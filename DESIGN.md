# Design System — CarLabra

## Design direction

The visual direction is inspired by premium automotive editorial websites, especially Ferrari-style digital presentation.

The site should feel:
- premium
- cinematic
- precise
- clean
- automotive
- slightly aggressive
- trustworthy

The design must not copy Ferrari directly. It should use similar logic:
- black/white contrast
- strong visual rhythm
- restrained accent color
- editorial sections
- premium spacing
- sharp details

## Brand color

Primary CarLabra accent:

`#9b5edb`

This is the main CarLabra purple/lavender color.

Use it as Ferrari uses red:
- sparingly
- for CTA buttons
- for active states
- for small highlights
- for icons/badges
- for hover states
- for section accents

Do not make the whole website purple.

## Color palette

### Primary

- CarLabra Purple: `#9b5edb`
- Black: `#000000`
- White: `#FFFFFF`

### Text

- Near Black: `#181818`
- Dark Gray: `#666666`
- Mid Gray: `#8F8F8F`
- Light Gray: `#D2D2D2`

### Optional supporting colors

- Deep Purple: `#6f35b5`
- Soft Purple: `#c8a8f0`
- Orange accent, only if needed for rare CTA contrast: `#F97316`

## Layout principles

- Alternate between dark cinematic sections and clean white editorial panels.
- Keep generous whitespace.
- Use strong black sections for hero/footer.
- Use white/light sections for form and explanatory content.
- Avoid clutter.
- Each section should have one clear purpose.

## Typography

Use modern sans-serif fonts.

Preferred fallback:
- Inter
- Arial
- Helvetica
- sans-serif

Typography principles:
- large bold hero heading
- clear section titles
- muted secondary descriptions
- uppercase small labels where appropriate
- strong but clean button labels

## Buttons

### Primary button

- background: `#9b5edb`
- text: white
- hover: darker purple
- border radius: minimal, 2–6px
- strong, wide, mobile-friendly

### Secondary button

- transparent or black
- white/purple border
- text white or purple depending on background

### External buttons

Gallery and Shop buttons should visually match the premium design.

## Header

Header should include:
- CarLabra logo if available
- text logo fallback
- navigation buttons:
  - Estimate
  - Gallery
  - Shop
  - Contact

Logo source:

`/public/assets/logoCarlabra.svg`

If missing, use text logo `CarLabra`.

## Hero

Hero should feel premium and automotive.

Include:
- brand name CarLabra
- strong headline
- short description
- CTA to calculator
- trust points/badges

Use dark background with possible subtle purple accent.

## Calculator section

The calculator must be clean, readable, and mobile-friendly.

Form fields should be:
- clear
- not crowded
- easy to tap on mobile
- consistent with premium design

## Footer

Footer should be dark, premium, and structured.

Include:
- company info
- services column
- working hours
- contact info
- social links
- copyright

Use black background, white/gray text, and purple accent.

## Do

- Use purple as a premium accent.
- Keep UI minimal.
- Keep strong contrast.
- Make mobile layout excellent.
- Use clean spacing.
- Keep automotive feeling.

## Do not

- Do not overuse purple.
- Do not create a cheap colorful UI.
- Do not use too many shadows.
- Do not use heavy border radius.
- Do not remove existing functionality.
